# Projects Repository

This is the parent repository containing all individual projects and development work.

## Repository Structure

This repository serves as a master container for all development projects, including:

- **Security & Network Projects**: Network monitoring, security platforms, DNS monitoring
- **Hardware Projects**: Arduino, ESP32, Marauder projects, PlatformIO-based development
- **Web Applications**: Various web-based tools and dashboards
- **Analysis Tools**: Investigation tools, forensic utilities, data analysis scripts
- **Infrastructure**: Docker containers, monitoring stacks, automation scripts
- **AI/ML Projects**: AI-powered tools and automation platforms

## Getting Started

Each project directory contains its own README with specific setup instructions.

### Prerequisites

Different projects may require:
- Python 3.x
- Node.js
- Arduino IDE / PlatformIO
- Docker & Docker Compose
- Various platform-specific tools
